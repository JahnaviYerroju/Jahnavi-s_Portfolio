GAN Defense Mechanism

Project Overview

Developed a defense mechanism using Generative Adversarial Networks (GANs) to combat adversarial attacks, specifically employing the Fast Gradient Sign Method (FGSM) on the CIFAR-10 dataset.

Implementation Details

Implemented the GAN architecture in Python with PyTorch and TensorFlow, enhancing model robustness. Evaluated accuracy improvements in a classifier against adversarial examples, demonstrating the effectiveness of GANs in mitigating adversarial effects.

Technologies Used

  • Python
  • PyTorch
  • TensorFlow
  • CIFAR-10 Dataset
  • Fast Gradient Sign Method (FGSM)
